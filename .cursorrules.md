# MindCalc Rules

## Tech Stack
- Framework: Next.js 14 (App Router)
- Language: TypeScript
- Styling: Tailwind CSS (Slate-950 background, Slate-100 text)
- Database: Supabase (Auth & Postgres)
- AI: Mistral AI (via SDK)
- UI Library: shadcn/ui (radix-ui based)

## Project Structure
- @/components: Reusable UI components
- @/lib: Logic and utilities (engine.ts, supabase client)
- @/hooks: Custom React hooks
- @/app: Routes and Layouts

## Coding Standards
- Use "use client" only when necessary (interaction/hooks).
- Prefer Server Components for data fetching.
- Use `lib/engine.ts` for math logic.
- Follow the established design system: Dark mode default.
